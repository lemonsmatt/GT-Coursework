To run code, open the folder in and IDE as a java project.
Build the project
Run the GW.java file's main method
A folder called output will contain all the data once it is done running

The code is based on the work of Enmao Diao who is cited in the paper. 
Their repo is located at https://github.com/dem123456789/Machine-Learning/blob/master/MDPs%20and%20Reinforcement%20Learning/ediao3-analysis.pdf
